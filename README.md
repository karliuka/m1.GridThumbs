# Magento GridThumbs
The Grid Thumbs extension adds a column with items images.
Adds a column with items images to make product grid more informative and simplify the process of products searching.

Installation
-------

Download the plugin and copy the 'app' and 'skin' folder to the root of your Magento installation.
